# Лабораторная работа №1

## Тема
Работа с Git и GitHub. Вариант 2b: Функции смены регистра строк.

## Описание программы
Программа принимает строку от пользователя и выводит её в верхнем и нижнем регистре.

## Функции
- to_upper(s) — переводит строку в верхний регистр
- to_lower(s) — переводит строку в нижний регистр

## Как запустить
python3 string_utils.py

## Пример работы
Введите строку: Привет Мир
Верхний регистр: ПРИВЕТ МИР
Нижний регистр: привет мир
